属性选择器
选择器 说明 示例
[attribute] 匹配包含给定属性的元素 $("div[name]") //匹配含有name属性的div元素
[attribute=value] 匹配给定属性是某个特定值的元素 $("div[name='test']") //匹配含有name属性是test的div元素
[attribute!=value] 匹配给定属性不是某个特定值的元素 $("div[name='test']") //匹配含有name属性不是test的所有div元素
[attribute*=value] 匹配给定属性是包含某些值的元素 $("div[name*='test']") //匹配含有name属性含有test值的div元素
[attribute^=value] 匹配给定属性以某些值开头的元素 $("div[name^='test']") //匹配含有name属性以test开头的div元素
[attribute$=value] 匹配给定属性以某些值结尾的元素 $("div[name$='test']") //匹配含有name属性以test结尾的div元素
E[attr~=val] 选择所有attr属性包括多个空格分隔的值，其中一个值等于“val”的E元素 E[attr|=val] 选择所有attr属性包含多个连字符-分隔的值，其中一个值以“val”开头的E元素
[selector1][selector2][selectorN] 符合选择器，需同时满足多个条件时使用 $("div[id][name$='test']") //匹配具有id属性并且name属性以test结尾的div元素
子元素选择器
选择器 说明 示例
:first-child 匹配所有给定元素的第一个子元素 $("ul li:first-child") //匹配ul 元素的第一个子元素li
:last-child 匹配给定元素的最后一个子元素 $("ul li:last-child") //匹配ul 元素最后一个子元素li
:only-child 匹配元素中为唯一的子元素 $("ul li:only-child") //匹配只含一个li元素的ul元素的li元素
:nth-child(index/even/odd/equation) 匹配其父元素下的第N个子元素或奇偶元素，这里index从1开始，而不是从0开始 $("ul li:nth-child(even)") //匹配ul中索引值为偶数的li元素
$("ul li:nth-child(3)") //匹配ul中第3个li元素
过滤选择器
简单过滤器
过滤器 说明 示例
:first 匹配找到的第一个元素,和选择器结合使用 $('tr:fist') //匹配表格的第一行
:last 匹配找到的最后一个元素，和选择器结合使用 $('tr:last') //匹配表格的最后一行
:even 匹配所有索引值为偶数的元素，索引值从0开始 $('tr:even') //匹配所有索引值为偶数的行
:odd 匹配所有索引值为奇数的元素，索引值从0开始 $('tr:odd') //匹配所有索引值为奇数的行
:eq(index) 匹配给定索引值的元素 $('tr:eq(1)') //匹配第二个行元素
:gt(index) $('tr:gt(2)') //匹配第二个及以上的行元素
:lt(index) $('tr:lt(2)') //匹配第二个及以下的行元素
:header 匹配如h1,h2,h3,h4...之类的元素 $(':header') //匹配所有标题元素
:not(selector) 去除所有与给定选择器匹配的元素 $('input:not(:checked)') //匹配没有被选中的input元素
:animated 匹配正在执行动画的元素 $(':animated') //匹配正在执行动画的元素
可见性过滤器
$("input:visible").val() //取得显示的input的值 $("input:hidden:eq(0)").val() //取得隐藏的索引值为0的input的值
内容过滤器
过滤器 说明 示例
:contains(text) 匹配包含给定文本的元素 $("li:contains('DOM')") //匹配含有DOM文本的li元素
:empty 匹配所有不包含子元素和文本的元素 $("td:empty") //匹配所有空单元格
:has(selector) 匹配含有选择器所匹配元素的元素 $("td:has(p)") //匹配含有< p >元素的单元格
:parent 匹配含有子元素或文本的元素 $("td:parent") //匹配不为空的单元格，即在该单元格还包含子元素或文本

$("span").parents("ul") 返回所有 < span > 元素的所有祖先，并且它是 < ul > 元素：
$("span").parentsUntil("div"); 返回介于 < span > 与 < div > 元素之间的所有祖先元素：
jQuery 遍历 - 后代
$("div").children() 返回每个 < div > 元素的所有直接子元素
$("div").children("p.1"); 返回类名为 "1" 的所有 < p > 元素，并且它们是< div > 的直接子元素：
$("div").children().each(function(){}) each方法逐个访问div的子节点
$("div").find("span") 返回属于 < div > 后代的所有 < span > 元素
$("div").find("*") 返回< div > 的所有后代
jQuery 遍历 - 同胞
$("h2").siblings(); 返回 < h2 > 的所有同胞元素
$("h2").siblings("p") 返回属于 < h2 > 的同胞元素的所有 < p> 元素
$("h2").next() 返回 < h2 > 的下一个同胞元素
$("h2").nextAll() 返回 < h2 > 的所有跟随的同胞元素
$("h2").nextUntil("h6") 返回介于 < h2 > 与 < h6> 元素之间的所有同胞元素
prev(), prevAll() 以及 prevUntil() 与上面的方法类似，只不过方向相反而已：它们返回的是前面的同胞元素
